### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/mrinalgupta24/Celabal-Technogies-Week3.git
    ```

2. Install dependencies:
    ```bash
    npm i
    npm install -D tailwindcss
    npx tailwindcss init
    ```

3. Start the application:
    ```bash
    npm start
    ```
